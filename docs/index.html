Domain Quote

Use this fast quote lookup to compare domain registration costs across currencies with optional discounts. While only a few popular extensions are displayed below, the library supports many more.

Need to adjust quoting globally? Provide a markup in your config (percentage or fixed USD) and the library will increase the base before discounts and VAT.

Install & usage guide is on the npm page.
